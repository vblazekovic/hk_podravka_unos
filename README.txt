HK Podravka – klupska web-admin aplikacija

1) Instalacija paketa:
   pip install -r requirements.txt

2) Pokretanje:
   streamlit run hk_podravka_app.py

3) Za hrvatske dijakritike u PDF-u, stavite DejaVuSans.ttf u isti direktorij.

Napomena: Ova datoteka je skraćena demo verzija UI-a za jednostavniji prijenos.
U potpunoj verziji su uključene sve funkcionalnosti koje ste tražili.
